# sistema-gestion-biblioteca-dotnet
Sistema de Gestión de Biblioteca desarrollado en C#, .NET y SQL Server utilizando arquitectura por capas. Permite administrar libros, usuarios y préstamos mediante operaciones CRUD y gestión de base de datos relacional.

# Sistema de Gestión de Biblioteca

## Descripción

Aplicación desarrollada en C#, .NET y SQL Server para la gestión de bibliotecas. Permite administrar libros, usuarios y préstamos mediante una arquitectura por capas que separa la lógica de negocio, acceso a datos y presentación.

## Tecnologías utilizadas

- C#
- .NET Framework
- SQL Server
- Arquitectura por capas
- Git y GitHub

## Funcionalidades

- Registro de libros
- Consulta y actualización de información
- Gestión de usuarios
- Préstamos y devoluciones
- Búsqueda de registros
- Operaciones CRUD

## Arquitectura

El proyecto está organizado en:

- Capa de Presentación
- Capa de Negocio
- Capa de Acceso a Datos
- Base de Datos SQL Server
